# IEPS-TOLPA Crawler

Course repository for **Searching and Extracting Data from the Web** (IEPS), University of Ljubljana, Faculty of Computer and Information Science.

## Repository structure

| Folder | Contents |
|--------|----------|
| `pa1/` | Programming assignment 1 — preferential web crawler targeting `github.com`, focused on image segmentation repositories. Includes the crawler implementation, database schema, and submission report. |
| `pa2/` | Programming assignment 2 — README extraction, segmentation, embeddings, and semantic retrieval over PostgreSQL/pgvector. |
| `pa3/` | Programming assignment 3 — RAG system (With Context / LLM-only) built on PA2 retrieval with Ollama. |
