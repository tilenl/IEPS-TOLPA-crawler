Naslov API strežnika, iz katerega UI bere podatke.
Preklopi med mock in real bazo (API query param `profile`). Zadnja izbira se shrani v brskalniku.
Omeji število vozlišč (manj = hitreje in bolj pregledno).
Filtrira grafe na izbrane domene (mora obstajati v bazi).
Filtrira tipe strani (CSV, npr. `HTML,DUPLICATE`; prazno = vse).
Pages = vozlišče je stran; Domains = vozlišče je domena (agregirano).
Load naloži graf iz baze; Fit centrira in približa graf v pogled.
Barve veljajo v načinu Pages; v načinu Domains so vsi node-i “DOMAIN”.
